程序基本结构

以下用红色标注的代码块，是你目前无需理解的内容。

我们给IDE打开，选择创建控制台应用，他会展示一个最基本的程序：

#include <iostream>
using namespace std;
int main(){
cout<<"Hello,World!";
//上面的句子输出了Hello,World!
}

  • 这#include <iosteam>意思是该程序包含了iostream库；

  • 这using namespace std;是指，接下来的代码块使用了命名空间std（它被包含在iostram库中）;

  • int main() {}是主函数，它是整个程序的入口。

  • cout<<"Hello,World!";是输出流函数，它把字符串Hello,World!输出到控制台上。

  • 绿色的一行字是注释，它不会被执行，可以用来说明程序某一部分的功能。

在这些语句中，只有int main() {}是必不可少的，每个程序都由int main() {}开始执行；

另外，每个语句的结尾，都必须加上一个分号，表示该语句执行完毕。

我们以后接触到的程序中，大多数都会用到iostream库，所以我们将主要通过改变int main() {}中的内容来调整程序的功能。

变量类型与声明

编程时需要用到各种变量来存储信息，以下是变量的类型：

  • int：整型，存储整数。

  • long int：长整型，存储特大整数。

  • float：浮点型，存储小数。

  • double：双精度浮点型，存储高位小数。

  • bool：布尔型，只能存储：true和false。

  • char：字符型，存储一个字符。

声明一个变量的语句是：<前缀> <数据类型> [<名称> (= 值)] ……

可以在一个语句中声明多个变量（使用逗号将变量隔开），也可以直接为它们赋值。

在Visual Studio中，变量的名称可以是中文、英文、沈从文、特立尼达和多巴哥文、数字以及他们的结合。需要注意的是，不可以用数字开头。

* 数据的表示法及其范围

这里的内容不是必要的，仅作了解即可。它可能帮助你更好地理解数据的存储方式。

  • int/long int：第一个二进制位存储正负号，0为正1为负。其余位将一个数字用二进制原码表示。

  • float/double：浮点型分为两部分，前半部分储存2的指数，后半部分储存尾数（二进制小数，1点……），他们相乘即是浮点数。
    当然同一浮点数可以有不同种表示，如1.5可以表示为0...01|00011...(2¹×0.75)，也可表示为0...10|000011...(2²×0.375)

  • char：通过二进制的方式存储一个ASCII码。

基本运算符

变量可以通过运算符来与其他变量和常量交互，组成表达式来表示一个新的值。

我们将所有运算符抽象为函数，便于理解：

普通的算术运算符

运算符表示法描述对应函数
+（加）a+b表示两个数相加\(f(a,b)=a+b\)
-（减）a-b表示两个数相减\(f(a,b)=a-b\)
*（乘）a*b表示两个数相乘\(f(a,b)=a×b\)
/（除）a/b表示两个数相除\(f(a,b)=a÷b\)
%（余）a%b表示a除以b的余数\(f(a,b)=a \ mod \ b\)

关系运算符

运算符表示法描述对应函数
>（是否大于）a>b如果a大于b，返回true，否则返回false\(f(a,b)=\left\{\begin{aligned}1 \ , \ a>b \\ 0 \ , \ a \le b \\\end{aligned}\right.\)
>=（是否大于等于）a>=b如果a大于等于b，返回true，否则返回false\(f(a,b)=\left\{\begin{aligned}1 \ , \ a \ge b \\ 0 \ , \ a < b \\\end{aligned}\right.\)
==（是否等于）a==b如果a等于b，返回true，否则返回false\(f(a,b)=\left\{\begin{aligned}1 \ , \ a=b \\ 0 \ , \ a \ne b \\\end{aligned}\right.\)
!=（是否不等）a!=b如果a不等于b，返回true，否则返回false\(f(a,b)=\left\{\begin{aligned}1 \ , \ a \ne b \\ 0 \ , \ a = b \\\end{aligned}\right.\)
<（是否小于）a<b如果a小于b，返回true，否则返回false\(f(a,b)=\left\{\begin{aligned}1 \ , \ a<b \\ 0 \ , \ a \ge b \\\end{aligned}\right.\)
<=（是否小于等于）a<=b如果a小于等于b，返回true，否则返回false\(f(a,b)=\left\{\begin{aligned}1 \ , \ a \le b \\ 0 \ , \ a > b \\\end{aligned}\right.\)

逻辑运算符

运算符表示法描述对应函数
&&（和）a&&b如果a，b都不是false，才返回true，否则返回false\(f(a,b)=\left\{\begin{aligned}1 \ , \ a \ne 0 \ 且 \ b \ne 0 \\ 0 \ , \ a=0 \ 或 \ b=0 \\\end{aligned}\right.\)
||（或）a||ba，b中有一个不是false，就返回true，否则返回false\(f(a,b)=\left\{\begin{aligned}1 \ , \ a \ne 0 \ 或 \ b \ne 0 \\ 0 \ , \ a=0 \ 且 \ b=0 \\\end{aligned}\right.\)
!（非）!a如果a为0或false，则返回true，否则返回false\(f(a,b)=\left\{\begin{aligned}1 \ , \ a=0 \\ 0 \ , \ a \ne 0 \\\end{aligned}\right.\)

应当注意的是，这里的==、<等，都是和“加减乘除”等同的，数学运算符号，和表示数字关系的符号“大于、小于”有本质区别。

一般的运算过程是，先算术，再关系，最后逻辑。算术又可分为先乘除(余)后加减。

赋值号、自增号和括号

赋值号的作用是，将赋值号后面的值（常量和变量都可以），拷贝给赋值号前面的变量。（当然，和运算符一起出现的时候，赋值号是最后才执行的。）

int main(){
    int a=6,b,c;
    c=a;
    b=c;
}

这个运算中，a的值被传递给c，c的值再传递给b。

有一种特殊的赋值号：在赋值号前加上一个普通算术运算符，表示先经过相应运算，再把结果赋值给左边的变量。
比如，x *= 3，相当于x=x*3。
类似地，x += 2相当于x=x+2等。

int main(){
    int a=6,b=1,c=2;
    c*=a;
    b+=c;
}

这个运算中，先进行c=c*a，再进行b=b+c。最终结果是，c=12，b=13。

在变量前或后添加“++”或“--”，相当于使这个变量增加1或减少1，即“x++”相当于“x=x+1”。
自增号的位置不同，功能也略有不同：自增号放在前面，则运算时先返回变量的值，再使自己加减1；放在后面，则先使自己加减1，再返回它的值。
（和运算符、赋值号同时出现的时候，自增号优先执行。）

int main(){
    int a=6,先增,后增;
    先增=++a;
    后增=a++;
}

这个运算的结果是，先增=7，后增=6。

括号的作用很好理解：不论何种运算，都要优先计算括号内的内容；内层的括号先算。

以下是例子：

int main(){
    int a=1,b=2,c=3;
    int d=6+( (a+b)*(c%b)+(b+c) );
}

这个运算的结果是，d等于6+(3×1+5)等于14。

总结

优先级：自增号>运算符>赋值号，运算符又分为算术>关系>逻辑，算术运算符又分为先乘除（余）后加减，同级从左到右算。

* 整数的按位运算

考试时这里不会出太难的内容，让你算的数不超过10，差不多得了。

我们回顾一下整数的存储方式：第一位是正负号，0为正，1为负，其余位用二进制原码表示数字。
于是，我们引入一组运算符——按位运算符，它通过二进制层面来操作整数。
如果只对数字进行操作，没有声明它的类型，它就默认是int(32位，4字节)。

运算符表示法描述对应函数
~（按位非）~a将整数用二进制表示，检查每个二进制位：如果是1则变成0，是0则变成1。\(f(x)=-(x+1)\)
<<（按位左移）a<<b将a的每一位都往左移b位，溢出就舍去，少位就补0。\(f(x)=x\cdot 2^b\) ，若不溢出
>>（按位右移）a>>b将a的每一位都往右移b位，溢出就舍去，少位就补0。\(f(x)=x\cdot 2^{-b}\) ，若不溢出
&（按位与）a&b检查两个数的每组对应位，两个位如果都是1则变1，有一个是0就变0。无
^（按位异或）a^b检查两个数的每组对应位，相同则变0，不同则变1。无
|（按位或）a|b检查两个数的每组对应位，有一个是1则变1，都是0则变0。无

这几个运算符的优先级从上到下依次降低，最上面的（按位非）优先算。
示例：
8^13=
(27个0)01000
^(27个0)10101
=(27个0)11101

循环和判断

循环

循环语句是指，给定一个条件表达式，只要条件满足，就不断执行其内部的语句，直到这个条件不成立（值为false）为止。
常用的循环有while和for两种：

while的用法是：while(条件){执行的语句}

for的用法是：for([先声明一个变量];[条件];[语句执行完后的操作]){执行的语句}

示例：

#include <iostream>
using namespace std;
int main(){
    int a=10;
    while(a>0){
      cout<<"a目前是"<<a<<endl;
      a--;
    }
}


#include <iostream>
using namespace std;
int main(){
    for(a=10;a>0;a--){
      cout<<"a目前是"<<a<<endl;
    }
}

看起来for要比while方便简洁。这两个程序的运行结果相同，你可以思考一下运行结果是什么。

在循环中，continue语句可以直接跳过本次循环，break语句可以直接结束循环。 

#include <iostream>
using namespace std;
int main(){
    int a=5;
    while(true){
      a++;
      if(a%2==0){continue;}
      if(a>15){break;}
      cout<<"a目前是"<<a<<endl;
    }
}

判断

判断语句是指：给定一个条件表达式，如果条件满足（值为true），就执行其内部的语句；否则，就不执行，或者执行另一段语句。

判断语句的表示法：if(条件){语句}
或者：if(条件){语句} else if(条件){另一段语句} else if …… else{第n段语句}

示例： 

#include <iostream>
using namespace std;
int main(){
    for(a=10;a>0;a--){
      if(a%2==0){
          cout<<"现在a是偶数："<<a<<endl;
        }
      else{
          cout<<"现在a是奇数："<<a<<endl;
        }
    }
}

还有一种多元判断语句switch，当表达式的可能取值很多时，不妨用switch语句来简化代码：
switch(表达式){
case <值1>...;break;
case <值2>...;break;
......
default:...;break;
}

示例： 

#include <iostream>
using namespace std;
int main(){
    int a;
    cout<<"请输入1-3里任意一个整数：";
    switch(a){
      case 1:cout<<"你输入了1";break;
      case 2:cout<<"你输入了2";break;
      case 3:cout<<"你输入了3";break;
      default:cout<<"？看不懂中国话是吧😅";break;  //default是指case的所有值都没取到时，执行的语句。这一句可以有，也可以没有
    }
}

* do-while语句

以下的内容很少会用到。

do-while语句是while语句的变体，它表示先执行一次代码块，接下来，如果满足条件，就循环下去，不满足则中断循环。
它的用法是，把do放在语句块前面，把while和条件放在语句块后面：
do{
...
}while(<条件>)

示例： 

#include <iostream>
using namespace std;
int main(){
  int x;
  cin>>x;
  do(){
    if(x>=60){cout<<"及格";}
    else{cout>>"不及格";x+=5;}
  }while(x>=60)
}

* 判断运算符

以下的内容知道也行，不知道也行

编程时，我们经常会用到这样的函数，输入一个值，如果满足一个条件就返回某一个值，不满足就返回另一个值，比如： 

int 最大值(int a,int b){
  if(a>b){return a;}
  else{return b;}
}

这样的表述有些麻烦，我们可以通过判断运算符来简化操作。
它的用法是：<条件>?<满足返回值>:<不满足返回值>
示例：

int 最大值(int a,int b){
  return a>b?a:b;
}

函数

函数可以把多个语句合并在一起并实现相应功能。函数可以让你的代码变得简洁，可读性也更强，也可以避免重复操作的麻烦。

函数有三个主要部分：返回值类型、内部语句和返回语句（return 值）。
这返回值是指，函数运算完毕后得到的结果，比如函数\(f(θ)=sinθ²\)，输入\(\sqrt \frac {\pi} {2} \)，则返回值就是1。

函数的声明与调用

函数的声明：<返回值类型> <名称> (参数1，参数2，...){代码}

函数可以单独调用：名称(参数...);或者作为表达式，如：a=b+王(b);

示例： 

#include <iostream>
using namespace std;
int 最大值(a,b){
  if(a>b){
      return a;
    }
  else{
      return b;
    }
}
int main(){
  int x,y;
  cin>>x>>y;
  cout<<"最大值是"<<最大值(x,y)<<;
}

另外，如果你的函数不需要返回任何值，那么返回值类型为void： 

#include <iostream>

简单循环判断

J1001 吴亦凡喝酒 难度：NOIP 入门组

传说李白出行，每次遇到酒家，总要先喝一半，再添两斗，若是离开后少于十斗，便会一饮而尽。
吴签在牢房里学李白喝酒，醒来什么都不记得了，只知道最后剩下x斗，且一开始不超过一千斗。问吴签一开始可能有多少斗酒。
标准：输入x(7≤x≤9)，每行输出开始时酒的可能值n。

这题解题方法很明确：设计一个运算循环并输出，当n>1000时，便跳出循环。

一、数学建模

\(f(x)=2(x-2)\) 。如果x小于1000，便输出x，再进行一次该运算。

二、代码实现

#include <iostream>
using namespace std;
int main(){
    int x;
    cin>>x;
    while(x<1000){
      x-=2;
      x*=2;
      cout<<x;
    }
}


J1002 数字倒换 难度：NOIP 入门组

大脚超市有一个神奇的电话本，凡是正着写进去的数字，都会倒着显示。

J1004 斐波那契 难度：NOIP 入门组

小明今天终于背下来“斐波那契”这四个字了。
不幸的是，他还把这个数列写错了。他写成的是：
$$\large a_n=\left\{\begin{array}{l} 1 \ , \ 1\le n\le 3 \\ a_{n-1}+2a_{n-2} , \ n \gt 3 \\\end{array}\right.$$标准：输入N（0<N<100），输出该数列的第N项。

这函数的特点是，要知道第n项，就必须知道n-1项及n-2项的值，而前几项又为定值。

一、数学建模

像上面的函数这样，某一项或几项为定值，其余每项都（由某个对应关系）与其他项有关的函数，叫做递归函数。
实现这个算法，只需要令它的返回值表达式调用它本身即可。

二、代码实现

#include <iostream>
using namespace std;
int 斐波那契(int n){
  if(n>=1 && n<=3){return 1;}
  else{
    return 斐波那契(n-1)+2*斐波那契(n-2);
  }
}
int main(){
  int a;
  cin>>a;
  cout<<斐波那契(a);
}

这个函数的执行过程是：首先被调用一次（斐波那契(a)），返回了斐波那契(a-1)+2*斐波那契(a-2)。
于是开始执行斐波那契(a-1)，它的返回值是斐波那契(a-2)+2*斐波那契(a-3)。……
这个过程将不断持续下去，直到所有的函数都返回1为止。

J1006 克里斯蒂安·哥德巴赫最想删除的视频.avi 难度：NOIP 普及组-

哥德巴赫给欧拉写过一封信，信里附带着质数猜想的描述，以及一段话：
“……我想出了一种简单的证法，几乎证明了它；但是我睡了一觉，把它忘了。……希望你能帮我证明。……”
——C.哥德巴赫
结果欧拉这个直男到死都不知道他在口胡。
三百年后的今天，人们可以用手机验证哥德巴赫猜想在某一区间内的正确性。
请你写一个程序，验证哥德巴赫猜想（2以上的偶数都能分解为两个质数之和）对3000以下的偶数成立。
标准：对大于2、小于3000的偶数，每行输出一个算式，表示出该数能分解成的两个质数。如果有多组解输出任意一组即可。若无法分解，则该行输出-1。
示例输出：
6=3+3
18=5+13

这道题有很多种方法来解决。我们可以用这样的解法：
对每个偶数n，我们检查区间 \( [3,\frac{n}{2}) \) 中的每一个奇数p，如果某一组p和n-p都是质数，那哥德巴赫猜想就对n成立。
特别地，4=2+2，它是唯一一个能被分解为两个偶质数的数。

一、数学建模

判断一个数是否是质数：对于一个数n，从2开始用各个质数除（我们事先不知道有哪些质数，所以不妨用所有的数来除）。
如果一直除到 \( \sqrt{n} \) 都没有能整除的，那么n就是质数。
为了使用函数\( f(x)=\sqrt {x} \) ，我们需要引入库cmath：#include <cmath>。

二、代码实现

#include <iostream>
#include <cmath>
using namespace std;
bool 质数(int a){
  for(int i=2;i<=sqrt(a);i++){  //这是cmath库的一个函数，\( sqrt(x)= \sqrt{x} \)
    if(a%i==0){
      return false;
    }
  }
  return true;
}
void 验证并输出(int a){
  bool 无解=true;
  for(int p=1;p<=a/2;p+=2){
    if(质数(p)&&质数(a-p)){
      cout<<  a << "=" << p << "+" << a-p <<endl;
      无解=false;
      break;
    }
  }
  if(无解){cout<<-1<<endl;}
}
int main(){
  cout<<"4=2+2"<<endl;
  for(int n=6;n<3000;n+=2){
    验证并输出(n);
  }
}

这种对所有可能符合要求的数据都进行运算的方法，叫做暴力搜索算法。

标准：输入数字n，以及n个正整数。接下来每行将各个数从大到小输出。

我们引入冒泡排序算法——冒泡排序是对一组数字排序的最简便方法之一。

这个算法的思想是：

  • 1、对于一排n个无规律的数字，进行如下操作：
    从左到右，如果前一个元素小于（大于）后一个，就将他们交换位置。重复这个操作（n-1次），则最小（大）的数字会“浮”到最右边；
  • 2、再执行一次这样的操作，这次“交换位置”执行了（n-2）次。
  • 3、再执行一次这样的操作，这次“交换位置”执行了（n-3）次。……

代码表示如下： 

#include <iostream>
using namespace std;
void 从小到大排序(int a[], int 长度){
    for (int i = 0; i < 长度 - 1; i++)
      for (int j = 0; j < 长度 - 1 - i; j++){
          if (a[j] > a[j + 1]){
              int 存储;
              存储 = a[j];
              a[j] = a[j+1];
              a[j+1] = 存储; //交换两个元素
          }
      }
}
int main(){
int n; cin>>n; int *a=new int[n];

for(int i=0;i<n;i++){
  cin>>a[i];
  }
从小到大排序(a,n);

for(int i=0;i<n;i++){
    cout<<a[i]<<endl;
  }
}

引用和指针

取地址和寻地址

取地址符（&）是一种运算符，它的运算结果是变量在内存中的位置。
寻地址符（*）则相反，它的运算结果是地址在内存中对应的变量。

#include <iostream>
using namespace std;
int main(){
  int x=666;
  cout<<&x;
  cout<<*(&x);
}

这个程序输出了x所在的地址以及666。

地址变量（指针）

在变量类型之后加一个“*”，则声明的变量将成为这个类型的地址变量（指针）。它将只能存储该类型变量的地址。
若地址变量中存储着变量x的地址，它就叫作“指向变量x的指针”。

#include <iostream>
using namespace std;
int main(){
  int x=666;
  int* y=&x;
  cout<<y;
  cout<<*y;
}

这个程序输出了x所在的地址以及666。

指针偏移 数组结构 指针与数组

指针的偏移是指，将指针加减一个整数，则它的内存地址将前移/后移这些位。如：

#include <iostream>
using namespace std;
int main(){
  int x=666;
  int* y=&x;
  cout<<y+0;
  cout<<y+2;
}

这将输出x的地址以及它的地址+2后的位置。

接下来我们观察数组的结构。当我们声明一个数组p，则p的值就是p的第一项p[0]的地址，p[1]的地址就是p+1，p[2]的地址就是p+2……
因此，我们使用p[x]访问数组元素，它就等同于*(p+x)。
相同地，如果我们想使用地址p右移x位地址对应的变量时，可以用p[x]来代替*(p+x)。

#include <iostream>
using namespace std;
int main(){
  int x=666;
  int* y=&x;
  cout<<y+0;
  cout<<y+2;
}

指针的应用

我们想定义一个函数，它可以交换数组中两个元素的值。

#include <iostream>
using namespace std;
void 交换(int a, int b){
  int 存储;
  存储=a;
  a=b;
  b=存储;
}
int main(){
  int x[]={1,2,3,4};
  交换(x[0],x[3]);
  for(int i=0;i<4;i++){
    cout<<x[i];
  }
}

你不会以为这程序会输出4231吧？😅实际上，形参的值被传递进函数后，实参并没有被修改。
但我们可以使用地址变量，对实参进行“降维打击”：

#include <iostream>
using namespace std;
void 交换(int* a, int* b){
  int 存储=0;
  存储 = *a;
  *a = *b;
  *b = 存储;
}
int main(){
  int x[]={1,2,3,4};
  交换(x,x+3);
  for(int i=0;i<4;i++){
  cout<<x[i];
  }
}

这函数中的两个形参是地址变量，它们的值就是数组中两个元素的地址。
因此，*a、*b等同于数组中的两个项x[0]和x[3]，改变它们，数组中的元素也会同步改变。
现在，输出值是4231了。
